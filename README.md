# Coffee Sales Dashboard (Excel)

An interactive Excel dashboard built to analyze coffee sales performance across time, countries, roast types, customer segments, and product sizes.  
The dashboard is fully dynamic and driven by advanced Excel formulas such as **XLOOKUP** and **INDEX**, without using Power BI or external tools.

---

## Dashboard Overview

This dashboard provides a comprehensive view of coffee sales trends, helping stakeholders quickly identify:
- Sales patterns over time
- Top-performing countries and customers
- Performance by roast type, loyalty status, and package size

The dashboard is designed with usability and business insights in mind.
